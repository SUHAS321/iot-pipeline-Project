# 🚀 IoT Data Processing & Storage Pipeline

## 📌 Overview
This project demonstrates a serverless IoT data pipeline using AWS.

## 🧩 Architecture
IoT → IoT Core → Lambda → DynamoDB + S3

## ⚙️ Technologies
- AWS IoT Core
- AWS Lambda
- DynamoDB
- Amazon S3
- CloudWatch
- Python

## 🔥 Features
- Real-time data ingestion
- Data validation & transformation
- Dual storage architecture
- Scalable serverless design
